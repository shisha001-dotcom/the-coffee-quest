📖 Hướng dẫn đổi banner
  1. Upload ảnh lên Google Drive
  2. Chuột phải vào file → Share → đổi thành "Anyone with the link"
  3. Copy link dạng https://drive.google.com/file/d/…/view
  4. Dán vào ô URL bên trên → bấm Xem trước để kiểm tra
  5. Bấm Lưu để áp dụng lên website
